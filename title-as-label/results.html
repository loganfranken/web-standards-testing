Using title as label Results

The following platforms were tested:

  • Firefox 38, Chrome 43, and Internet Explorer 11 with NVDA 2015.2 on Windows 8.1
  • Firefox 38, Chrome 43, and Internet Explorer 11 with JAWS 16.0.3048 on Windows 8.1
  • Safari with Voiceover on iOS 8.3

On all platforms, the results were the same:

  • input and label: Reads label
  • label hidden with CSS: Reads label
  • No label: Doesn't read label
  • No label with title attribute: Reads title attribute
  • No label with placeholder attribute: Reads placeholder attribute
  • No label with aria-label attribute: Reads aria-label attribute
  • Implied label: Doesn't read label

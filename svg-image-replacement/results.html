Using Image Replacement with Icon Fonts Results

View Tests

The following platforms were tested:

  • Firefox 41, Chrome 45, and Internet Explorer 11 with NVDA 2015.3 on Windows 7
  • Firefox 41, Chrome 45, and Internet Explorer 11 with JAWS 16.0.4350 on Windows 7

On all platforms, the results were the same: "Website Logo" was always read. In cases where an img element was embedded directly, the word "Graphic" was read before "Website Logo."

HTML5 Header Hierarchy Tests

View Tests

The following platforms were tested:

  • Firefox 41, Chrome 46, and Microsoft Edge 20 with NVDA 2015.3 on Windows 10

With the following results:

  • HTML5 heading hierarchy not read properly in NVDA (i.e. nested h1s read as h1s)
  • Heading levels not read before reading header in Edge
  • Navigating by headers (using number keys) does not respect HTML5 document outline (i.e. pressing 1 in the HTML5 hierarchy example cycles through all headers)
